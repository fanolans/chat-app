## Features
- One on One Chat
- Group Chat

## Tech Stack
Flutter, ZIM SDK, & Zegocloud Kit

## Run Locally
Clone the project

```bash
  git clone https://github.com/fanolans/chat-app.git
```

Go to the project directory

```bash
  cd project-folder
```

Install dependencies

```bash
  flutter pub get
```

Start the app

```bash
  flutter run
```
